# java-spring-boot
Java Spring Boot API

with CRUD function

##GET
```
/api/v1/student
```
##Post
```
/api/v1/student
```
__Request Body__
```
{
    "name":"Jonny",
    "email":"jonny@gmail.com",
    "dob":"2002-12-17"
}
```
##Put
```
/api/v1/student/id?name=soleh
```
##Delete
```
/api/v1/student/id
```
